# QXO Inc. Business Combination Announcement Website

A professional, responsive website showcasing QXO Inc.'s official SEC Form 425 filing regarding a significant business combination.

## Project Information

**Company:** QXO Inc.  
**CIK:** 1236275  
**Filing Date:** April 27, 2026  
**Filing Type:** Form 425 - Business Combination Communication  
**SEC Document:** tm2612209d4_425.htm  
**Headquarters:** Greenwich, Connecticut  
**Industry:** Building Products Distribution  

## Overview

This website serves as an informational hub for the public announcement of QXO Inc.'s business combination, as required by SEC regulations. The site provides easy access to official filing documents and related information.

## Features

- **Professional Design:** Modern, clean interface with gradient backgrounds and smooth animations
- **Fully Responsive:** Optimized for desktop, tablet, and mobile devices
- **SEC Compliance:** Links directly to official SEC EDGAR filing documents
- **Accessible:** WCAG-compliant markup and semantic HTML
- **Fast Loading:** Optimized CSS and JavaScript with minimal dependencies
- **Print-Friendly:** Professional print styling for documentation
- **Navigation:** Smooth scrolling between sections with sticky navigation bar
- **SEO Optimized:** Proper meta tags and structured content

## Sections

1. **Navigation Bar** - Sticky header with navigation links
2. **Hero Section** - Eye-catching banner with call-to-action buttons
3. **Announcement** - Official filing information and key details
4. **Details** - Information about Form 425 and QXO
5. **SEC Filing** - Direct link to the official SEC EDGAR document

7. **Contact & Resources** - Additional information and SEC resources
8. **Footer** - Company information and links

## Getting Started

### Local Development

1. Clone this repository:
   ```bash
   git clone https://github.com/zfaria522-wq/qxo-announcement-site-.git
   cd qxo-announcement-site-
   ```

2. Open `index.html` in your web browser or use a local server:
   ```bash
   python -m http.server 8000
   ```

3. Visit `http://localhost:8000` in your browser
